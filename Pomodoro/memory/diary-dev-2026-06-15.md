# 日记功能开发 - 2026-06-15

## 目标
在 pomodoro-wpf 中新增"日记"模块：左栏当月日历 + 本月/本周目标，右栏日记编辑器。

## 实现

### 新增文件
- `DiaryEntry.cs` - 数据模型（Date, Content, MonthGoal, WeekGoal）
- `DiaryView.xaml` - 两栏布局：左 320px 日历 + 目标卡片，右 RichTextBox 编辑器
- `DiaryView.xaml.cs` - 月历构建、日期选择、CRUD、JSON 持久化

### 修改文件
- `MainWindow.xaml` - 加 xmlns:local + `📝 日记` 侧栏按钮 + DiaryView 视图区
- `MainWindow.xaml.cs` - 加 SidebarDiary_Click + 所有侧栏 handler 加 DiaryBorder/DiaryView 状态复位

### 功能
- 月历网格：7列6行，自动填充，周日起始，周日红色，今天高亮黄色
- 有日记的日期显示红点指示器
- 点击日期选中（红色边框），右侧加载日记内容
- RichTextBox 支持多行编辑
- 本月目标（黄色卡片）+ 本周目标（绿色卡片），保存时自动同步到同月/同周所有条目
- 前一天/后一天/今天 导航按钮
- 数据持久化到 `%APPDATA%/Pomodoro/diary.json`
- 前后翻月自动切换月份

### 参考来源
WPFMonthView（yiyan127），.NET Framework 4 月历控件库。采用了月历布局算法（DayOfWeek 计算 startOffset），但整体重写为 .NET 8 适配。
